# Deployment Checklist

## Before deploy

- `npm install` passes
- `npm run lint` passes
- `npm run build` passes
- No broken image paths
- No broken resume link
- All CTAs work
- LinkedIn opens correctly
- StriveAI link opens correctly
- Email link uses `mailto:`
- Mobile layout tested
- Metadata added
- Favicon added
- `.env.example` exists
- `.gitignore` protects local env files

## After deploy

- Production URL opens
- Mobile version works
- Social preview works
- Resume download works
- Contact links work
- No console errors
- Lighthouse basic check completed
